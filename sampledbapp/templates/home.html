Welcome to SampleDB

To get started:

  1. Configure a project.
  2. View, edit, import and export your samples.
  3. View your imported and exported files.
  4. Validate, correct, and confirm your changes prior to commit.
  5. Check you notifications in the top right.
  6. Check your task progress here and in the top right.

Finding existing samples:

  • Use the Project samples to view, edit and export samples by project.
  • Use the Audit search to search samples by Sample ID, Freezer, Box, or Tissue Bank Reference. You must be an Auditor to view this page.

General info:

  • Each Project Lead has a Group with various Projects.
  • Samples are grouped into Projects.
  • Each sample is linked to a Project and Study
  • Sample IDs must be unique per study.
  • Samples can be downloaded to excel.
  • Excel files with new and edited samples can be uploaded.
  • Edited excel files must contain the pk field (primary key). This is used for identify which sample you editing.
  • Changes to the DB are validated prior to committing. Failures must be manually edited and resubmitted.
  • Export, import, and validation checks are run offline as Tasks.
  • User management, including assigning users to groups, is managed in the admin interface
  • If you require access to a project, please contact the PI responsible.
